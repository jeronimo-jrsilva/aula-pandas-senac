# 🧪 Revisão de Pandas: Limpeza e Transformação de Dados
Atividade prática da aula de **Analista de Dados (UC4)** do Senac.

## 🎯 Objetivo da Atividade
Praticar conceitos fundamentais de engenharia de dados com a biblioteca Pandas:
- Tratamento de valores ausentes (nulos) e duplicados.
- Remoção estatística de outliers usando o método IQR (Intervalo Interquartil).
- Normalização de escala (Min-Max).
- Padronização de variáveis categóricas e criação de indicadores.
- Cruzamento e empilhamento de bases de dados (`pd.merge` e `pd.concat`).

---

## 🚀 Como Executar o Projeto

### 1. Clonar ou Baixar o Repositório
Abra o terminal do seu computador e execute:
```bash
git clone https://github.com/jeronimo-jrsilva/aula-pandas-senac.git
```
*(Nota: Substitua pelo link do repositório caso o professor altere o endereço)*

### 2. Criar seu Notebook ou Script
1. Abra o **VS Code**.
2. Abra a pasta clonada (`File` -> `Open Folder...`).
3. Crie um notebook novo (ex: `pratica.ipynb`) ou script (`pratica.py`).
4. Abra o arquivo **`pratica_pandas_limpeza.md`** ao lado para acompanhar os enunciados e copie os blocos de código iniciais (Passo 0 e Passo 1) para rodar e gerar seu dataset exclusivo.
5. Complete os desafios dos **Passos 2 ao 6** implementando as resoluções propostas.
